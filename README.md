👥 Client-Service Manager
A robust CRM solution for managing client interactions and service lifecycles.

Client-Service Manager is designed to streamline the "Client-to-Service" relationship. It provides a centralized hub for tracking client profiles, service agreements, and support history with high data integrity.

🚀 Key Features
👤 Client Lifecycle Management: Full CRUD operations for client profiles with nested contact management.

🛠️ Service Catalog: Define service types, pricing models, and availability.

📝 Contract Tracking: Manage active service agreements and automated renewal alerts.

📊 Audit Logging: Comprehensive history of all changes to client data.

🛠️ Tech Stack
Backend: Spring Boot, Spring Data JPA

Database: PostgreSQL (with Flyway for schema versioning)

Validation: Hibernate Validator & custom constraints

Security: Spring Security (JWT-based)
